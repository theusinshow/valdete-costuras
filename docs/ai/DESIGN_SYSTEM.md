## Visual Direction
> **Updated per DEC-014 — aligned to the official brand manual (`identidade_visual/`).**

**Costura Artesanal** — warm, handcrafted, trustworthy. Linho (warm off-white) canvas, Carvão ink, the Vermelho Valdete thread as the color voice, softened by coral/rosé details. Generous whitespace and a large typographic hero remain, but the tone is affectionate rather than clinical. Brand signatures: the dotted-circle "V" monogram, the script wordmark, and the traço-costura (stitch dashes). Support tones (coral/rosé) are decorative only.

Contrast notes (verify in QA):
- `--text` `#2C2523` on `--bg` `#FBF6F2` → ~13:1 (AAA). ✅
- `--text-muted` `#6E625E` on `--bg` → ~4.7:1 (AA normal). ✅

- Coral/Rosé: decoration only — never text or long copy.

## Typography
> **Updated per DEC-014** — adopts the brand manual's three type voices (supersedes DEC-012 Petrona + Hanken Grotesk).
- Display/headings: **Cormorant Garamond** (elegant serif) weights 400–700, tracking ~-0.01em.
- Body/UI: **Jost** (clean geometric sans) weights 300–600.
- Script signature: **Alex Brush** — the `.wordmark` logo only. Never body, titles, or UI.
- Load via `next/font/google`, subset `latin`, `display: swap`.
- Fallbacks: `"Cormorant Garamond", Georgia, serif` (display), `"Jost", system-ui, sans-serif` (body), `"Alex Brush", "Segoe Script", cursive` (script).

Type scale (rem, ~1.25 modular, dramatic hero):

| Role | Size (desktop) | Line-height | Weight | Tracking |
|---|---|---|---|---|
| display / H1 (hero) | 3.5–4.5rem (clamp) | 1.05 | 700 | -0.02em |
| H2 | 2.25rem | 1.15 | 600 | -0.01em |
| H3 | 1.5rem | 1.2 | 600 | -0.01em |
| H4 / eyebrow | 0.875rem | 1.2 | 600 | 0.08em, uppercase |
| body-lg | 1.125rem | 1.6 | 400 | 0 |
| body | 1rem | 1.6 | 400 | 0 |
| small / caption | 0.875rem | 1.5 | 400 | 0 |

Hero uses `clamp()` for fluid scaling. Section titles: tight tracking, high weight.

## Spacing Scale
Base 4px. Scale: `4, 8, 12, 16, 24, 32, 48, 64, 96, 128`.
- Component padding: 16–24.
- Section vertical rhythm: 96 desktop / 64 mobile (generous, minimal-premium).
- Card padding: 24.

## Containers and Grid
- Content container: `max-width: 1200px`, side padding 24 (mobile) / 32 (desktop).
- Some sections go **full-bleed** (final CTA band, hero background) with inner container.
- Grid: 12-col conceptual; implemented with CSS grid/flex utilities (Tailwind).
- Services grid: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-3`.

## Radius and Shadows
- Radius: `--radius-sm: 6px`, `--radius: 10px`, `--radius-lg: 16px`.
- Shadows: minimal. `--shadow-sm: 0 1px 2px rgba(20,20,20,.06)`; hover lift `0 6px 20px rgba(20,20,20,.08)`. Prefer borders/hairlines over heavy shadow.

## Essential Components
- **Navbar** (sticky): wordmark + anchor links (desktop) + WhatsApp button; mobile: wordmark + WhatsApp icon.
- **Footer**: brand, contact, address, hours.
- **Buttons**: primary (accent bg, white text), secondary (outline, dark text), WhatsApp (accent + WA icon). Min height 44px (tap target).
- **Links**: `--accent-strong`, underline on hover, visible focus.
- **Containers / Section wrapper**: consistent padding + optional eyebrow label.
- **Section header**: eyebrow (uppercase) + H2 + optional lead line.
- **Service card**: icon + title + short description; hover lift + accent hairline.
- **Step / process item**: number, title, text.
- **Placeholder block**: for gallery/testimonials ("em breve") — must look intentional, not broken.
- **Floating WhatsApp button** (mobile).

## Responsive Principles
- Mobile-first. Breakpoints: `sm 640`, `md 768`, `lg 1024`, `xl 1280`.
- Fluid type via `clamp()` for hero and H2.
- Tap targets ≥44px. No horizontal overflow. Test 320px width.

## Asset Guidelines
- SVG icons for services (line icons, 1.5px stroke, match `--text`).
- Wordmark as text now; swap for logo SVG when finalized.
- Photos (future) via `next/image`, meaningful alt, before/after clearly labeled.

## Reference Principles
From "Costura Moderna": keep high contrast, whitespace, restraint. Avoid: stock-photo clichés of sewing, generic template look, heavy gradients, more than one accent color. The red is the only color voice.
